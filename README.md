# 📌 Habit Tracker (C Project)

A simple **console-based Habit Tracker** built using the C programming language.  
This project helps users track daily habits for a week and monitor their consistency.

---

## 🚀 Features

- Add new habits  
- Mark habits as completed (day-wise)  
- View all habits with status  
- Generate weekly progress report  
- Save data to file  
- Load saved data automatically  

---

## 🛠️ Technologies Used

- C Programming
- File Handling
- Structures
- Arrays

---

## 📂 Project Structure

HabitTracker/
│── main.c
│── habits.txt
│── README.md

---

## ▶️ How to Run

### Compile
gcc main.c -o habit

### Run
./habit

---

## 📖 How It Works

1 -> Add Habit  
2 -> Mark Habit as Done  
3 -> View Habits  
4 -> Weekly Report  
5 -> Save & Exit
